## Run Locally

Open `index.html`, or serve locally:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## GitHub Pages

Push repo to GitHub, then enable Pages from the `main` branch root.

```bash
git branch -M main
git add .
git commit -m "init: manual QA portfolio"
git remote add origin https://github.com/Mruunalll/manual-qa-portfolio.git
git push -u origin main
```
